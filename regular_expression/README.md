# 正则表达式

## 元字符

| 元字符 |               说明                |
| :----: | :-------------------------------: |
|   .    | 匹配 除换行符('\n')以外的任意字符 |
|   \w   |   匹配 字母、数字、下划线、汉字   |
|   \s   |         匹配 任意的空白符         |
|   \d   |             匹配 数字             |
|   \b   |       匹配 单词的开始货结束       |
|   ^    |         匹配 字符串的开始         |
|   $    |         匹配 字符串的结束         |

## 重复限定符

| 语法  |    说明    |
| :---: | :--------: |
|   *   |    >=0     |
|   +   |    >=1     |
|   ?   |    0,1     |
|  {n}  |     n      |
| {n,}  |     n+     |
| {n,m} | >=n && <=m |

## 分组

例如, 重复多次ab: `^(ab)*`

## 转义

例如, 匹配(ab)开头: `^(\(a\))*

## 条件或

`^(130|131|132)\d{8}$`

## 区间

[0-9], [A-Z]

## 贪婪和惰性

贪婪匹配: `.*`

惰性匹配: `.*?`
